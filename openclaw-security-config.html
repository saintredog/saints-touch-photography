1 · WhatsApp Channel Security

4 · How to Restore These Settings

If you ever need to re-apply these settings from scratch (reinstall, new machine, etc.):

  1. Open the OpenClaw TUI and ask Claw to apply the WhatsApp security config.
  2. Or run: openclaw config edit and paste the values from Section 1 manually.
  3. Set channels.whatsapp.dmPolicy → "allowlist"
  4. Set channels.whatsapp.allowFrom → ["+1", "+973"]
  5. Set channels.whatsapp.debounceMs → 3000
  6. Confirm channels.whatsapp.groupPolicy is "allowlist"
  7. Confirm the two bindings in Section 2 are present.
  8. Confirm photo_marketing_agent tool allow/deny list matches Section 3.
  9. Restart the gateway: openclaw gateway restart
